# philip-trauner.me

## Dependencies
```bash
sudo apt-get install imagemagick zopfli fswatch
```

## Install
```bash
git lfs install
git clone https://github.com/PhilipTrauner/philip-trauner.me.git
cd philip-trauner.me
make install
```

## Run
```bash
poetry run python3 app.py
```
